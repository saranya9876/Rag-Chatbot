# RAG Chatbot 🤖

A Retrieval-Augmented Generation system using:

- ChromaDB (Vector Database)
- FastEmbed (Embeddings)
- LM Studio (Phi-3 model)
- Streamlit (UI)

## Features
- Multi-PDF support
- Semantic search
- Context-aware answers
- Chat interface
